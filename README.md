🏠 House Price Prediction using Machine Learning

📖 Overview
This project predicts house prices using Multiple Linear Regression based on features like area, bedrooms, bathrooms, age, and location score.

🎯 Objective
To build a predictive model that estimates house prices accurately.

📊 Dataset
Custom dataset created with the following features:
- Area
- Bedrooms
- Bathrooms
- Age
- Location Score

⚙️ Algorithm Used
Multiple Linear Regression

📈 Results
- High R² Score indicating good accuracy
- Low Mean Absolute Error (MAE)

💻 Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

🚀 Future Scope
- Use advanced models like Random Forest
- Deploy as web application
